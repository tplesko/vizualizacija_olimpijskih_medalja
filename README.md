# Olympic Medal Visualization

Interactive data visualization of Olympic Games history from 1896 to 2016.

## About
This project allows users to explore and analyze Olympic medal data 
through dynamic visualizations and historical trends.

## Features
- Animated world map showing medal counts by country and year
- Play/pause animation controls
- Country detail view
- Filtering by year, medal type, and sport

## Technologies
- JavaScript
- D3.js (Data-Driven Documents)
- HTML5 & CSS3
- GeoJSON (world map rendering)
- CSV datasets (athlete_events.csv, olympic_medals_detailed.csv)

## Data Sources
Data loaded locally from CSV files covering Olympic Games 1896–2016.
No external APIs or databases used.
